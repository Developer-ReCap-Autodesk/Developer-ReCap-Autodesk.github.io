What is reality data?

To understand what the ReCap Photo API can do, we first need to understand 'What is reality data?'

Reality data is physical object and spatial information captured by sensors, such as laser scanners and photography. Whether youâ€™re using a terrestrial scanner, a depth camera, a digital SLR or a smartphone, ReCap software works with data captured from a range of devices. ReCap software interprets four types of reality data, depending on the capture technology used: Laser Point Cloud, Real View, Photo Mesh, and Photo Point Cloud.

Laser Scanning

  • Laser Point Clouds
    The collection of points in 3D space resulting from scanning an object using a laser scanner. Each point is defined by its X, Y, Z coordinates, to which is attached an intensity value, and, if available, RGB color and additional information.
  • Real Views
    Photo-realistic 360Â° panoramas from terrestrial scanners. Each pixel of the panorama includes depth information (X, Y, Z) from the laser scanning that ensures highly accurate and trustworthy data information.

Photography

  • Photo Meshes
    A photo-textured 3D model whose polygons and texture information have been extracted from a series of adjacent source photos by photogrammetry (a stitching/registration and meshing process converting 2D images to 3D mesh models).
  • Photo Point Clouds
    The collection of points in 3D space by scanning an object using a series of adjacent source photos through the same photogrammetry process used for Photo Meshes. Each point in space comes with an associated RGB color and additional information.

What exactly does the ReCap Photo API do?

AutodeskÂ© ReCapâ„¢ Photo provides a WEB service to create textured mesh from a set of photos, and can request an automatic 3D calibration. This REST API provides a similar service as the Autodesk ReCap 360 WEB application.

Using that API, you can develop a Mobile, Desktop, or WEB application. Your application will need an oAuth 1.0a key pair to access the API, and the customer using the application will give authorization to your application to access his resources by logging on his Autodesk 360 account.

The API comes as a (server-side) REST API that allows you to translate photos into a textured' mesh in the file format of your choice. The data extracted from the original photos are the 3d geometry and the texture of the model.

Getting started

Please consider Memento as your preferred tool for your sculpting projects. Not only you can process your photos and create high res 3D models but you can also clean and heal the mesh, and prepare it for 3D printing or web publishing.

  1. Make sure that the photogrammetry technology gives satisfactory results for the use case.
    Using the generic Autodesk ReCap 360 WEB application, you should be able to create a 3D models.
  2. Discuss the use case and the business case with an Autodesk representative
    You should discuss the use case, the type of application, and the business model with an Autodesk representative. From this discussion, Autodesk can make a proposal for the number of cloud credits to be applied to this use case per project. Drop an email to Dominique Pouliquen to start the discussion at ReCap Photo API
  3. Subscribe to Autodesk Developer Network program (aka ADN)
  4. Receive all the necessary ReCap Photo WEB API credentials using the form below, and resources to start the development
  5. Get support from the ADN team on using the API.
