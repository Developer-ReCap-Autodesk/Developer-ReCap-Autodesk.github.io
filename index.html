ReCap Photo API

Welcome to the Autodesk ReCap Photo WEB API Developer Center.

Autodesk© ReCap™ Photo provides a WEB service to create textured mesh from a set of photos, and can request an automatic 3D calibration. This REST API provides a similar service as the Autodesk ReCap 360 WEB application.

Using that API, you can develop a Mobile, Desktop, or WEB application. Your application will need an oAuth 1.0a key pair to access the API, and the customer using the application will give authorization to your application to access his resources by logging on his Autodesk 360 account.

Getting started.

Accessing ReCap Photo WEB API

  1. Make sure that the photogrammetry technology gives satisfactory results for the use case.
    Using the generic Autodesk ReCap 360 WEB application, you should be able to create a 3D models.
  2. Discuss the use case and the business case with an Autodesk representative
    You should discuss the use case, the type of application, and the business model with an Autodesk representative. From this discussion, Autodesk can make a proposal for the number of cloud credits to be applied to this use case per project. Drop an email to Dominique Pouliquen to start the discussion at ReCap API
  3. Subscribe to ADN (i.e. Autodesk Developer Network)
  4. Receive all the necessary ReCap Photo WEB API credentials using the form below, and resources to start the development (i.e. help & samples).
  5. Get support from the ADN team on using the API.
